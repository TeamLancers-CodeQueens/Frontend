ABOUT US

Code Queen is a form of educationinitiated by the Educating the Children(ETC).
ETC is a UK registered charity that focuses on empowering youngwomen
through educating and equipping girls with skills for employment.
ETC launched Code Queen, a all female coding boot camp mainly to
skill them with coding skills in web development and soft skills it
has successfully run over 4 cohorts and quaduated a number of ladies
and this was done physically. But due to the covid-19 breakout it was
then condcted online since 2020.
